# 💰 Expense Tracker with Charts

A web-based Expense Tracker built with **ASP.NET Core MVC (.NET 8)** that helps users manage daily expenses, organize them into categories, and gain insights through interactive charts. The application also includes income and budget management features for better financial planning.

## 📌 Overview

This application provides a simple and efficient way to track personal finances by allowing users to record expenses, manage categories, monitor budgets, and analyze spending trends with visual reports.

---

## ✨ Features

- Secure user authentication
- Expense management (Create, Read, Update, Delete)
- Category management
- Income tracking
- Monthly budget management
- Date-based expense filtering
- Interactive dashboard
- Automatic savings calculation
- Visual reports using Chart.js

---

## 🛠️ Technologies Used

- **Backend:** ASP.NET Core MVC (.NET 8), C#
- **Frontend:** Razor Views, HTML5, CSS3, Bootstrap
- **Database:** SQL Server, Entity Framework Core
- **Charts:** Chart.js
- **Architecture:** MVC (Model–View–Controller)
- **IDE:** Visual Studio 2022
---

## 📊 Dashboard

The dashboard provides an overview of:

- Total Income
- Total Expenses
- Savings
- Category-wise Expense Distribution
- Monthly Spending Trends
